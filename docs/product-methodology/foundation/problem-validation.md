# 문제 검증 (Problem Validation)
> 유형: foundation · 2026-07-21 · 방법: 5Why + 웹 리서치(실제 목소리) + 직접관찰

## 5Why (왜 못 찾나)
1. 왜 못 찾나? → 원하는 "백프린팅만 노란 + 기능성 + 내 사이즈" 티가 검색·필터로 안 걸림
2. 왜 안 걸리나? → 검색이 색을 **옷 전체**로만 인식, "프린팅색=노랑/바탕=흰" **조합**을 표현할 수단이 없음
3. 왜 조합이 안 되나? → 상품 데이터가 색·프린팅위치·소재·기능성 같은 **구조화된 속성**으로 존재하지 않음(제목·이미지에 파묻힘)
4. 왜 확인이 오래 걸리나? → 검색결과가 **대표 썸네일 1장**뿐 → 등판·소재·사이즈를 상품마다 상세페이지 들어가 확인
5. **근본 원인**: 상품이 "말로 조합해 검색할 수 있는 속성 데이터"로 구조화돼 있지 않고, 검색이 키워드 매칭에 머무름. → **속성 구조화 + 시맨틱(말로) 검색**이 해법.

## 직접관찰 (PM 본인, n=1 — 가장 강한 육성)
> 노란 클라이밍화 코디용으로 "백프린팅만 노란" 티를 네이버·무신사에서 필터·검색으로 찾음 → 이상한 것만 추천, 대표사진만 보여 일일이 상세 진입, 사이즈·질감·기능성 매번 확인 → **4시간 쓰고 원하는 티 못 찾음.**

## 기존 대안 (상세는 market-and-competition 예정)
- **직접경쟁**: SPIRI7(클라이밍 가이드+상품 큐레이션 커머스) — 단 "말로 찾는 검색"은 아님.
- **공급/브랜드**: 슬랩스틱·클라이맥스·쏘엠·온사이트·문 등 자체 그래픽 티 브랜드 다수(→ 경쟁 아닌 **카탈로그 소스**).
- **범용**: 네이버쇼핑·무신사 검색/필터(←실패 지점), 직구, 중고.

## 검증 판정 (4체크)
| 체크 | 판정 | 근거 |
|---|---|---|
| ① 겪는 사람 충분? | 🟢(모수)/🟡(육성) | 인구 50만·성장세. 단 "백프린팅 못 찾아 고생" 다수 육성은 미확보 |
| ② 이미 시간·돈 쓰나? | 🟢 | 직구·중고·현장구매 우회 + **본인 4시간 투입** |
| ③ 못 찾아 포기·우회? | 🟢(본인)/🟡(웹) | 본인 4시간 실패=명확. 웹 개인 육성은 약함 |
| ④ 기존 방식 명시 불만? | 🟢(본인)/🟡(웹) | 본인 육성 강함. 웹은 창업자·언론 중심 |

**종합**: 직접관찰(n=1)로 문제는 **강하게 유효**하나, 일반성은 아직 n=1 → **팀이 클라이머 3~5명 인터뷰로 확대 검증** 필요(필수 산출물과 연계). 문제 재정의: "클라이밍 그래픽 티가 **없다**"가 아니라 "**흩어져 있고 속성으로 검색이 안 돼 발견·비교가 안 된다**".
