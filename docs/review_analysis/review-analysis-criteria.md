# 리뷰 분석 기준서

> 클라이밍 프린팅 티 발견 검색(search-by-llm) — 상품 리뷰 분석 기준 문서

---

## 기본 원칙

1. **광고 리뷰의 완벽한 판별은 불가능하다고 전제**하고, 명확한 제외 기준으로 거른 뒤 남은 리뷰만 분석한다.
2. **별점은 점수 산정에 사용하지 않는다.** 단, 1~2점 리뷰는 광고일 확률이 낮은 순도 높은 데이터이므로 하자·반품 사유 수집용으로 우선 열람한다.
3. **모든 점수에는 근거 숫자(언급 건수 n)를 병기**하고, 표본이 부족하면 점수 대신 "데이터 부족"으로 표기한다.

---

## 수집할 데이터

1. 리뷰 텍스트, 사진, 작성일, 별점
2. 한달사용리뷰 여부, 재구매 뱃지
3. 구매 옵션 (색상/사이즈)
4. 네이버 구조화 평가 항목 (사이즈: 정사이즈/커요/작아요)
5. 리뷰 속 신체 정보 (키/몸무게)

---

## 제외 기준

1. **글자수 15자 미만 + 제품을 평가할 수 있는 구체적인 측면의 속성이 없음** — 길어도 정보가 없으면 제외, 짧아도 정보가 있으면 유지.
2. **같은 사진 재사용 리뷰** — 이미지 해시(pHash) 비교를 통해 판별.
3. **같은 텍스트 복붙 리뷰** — 사진뿐 아니라 텍스트 중복 검사를 통해 판별.
4. **대가성 협찬/체험단 리뷰**
5. **제품에 대한 내용이 전혀 없이 배송·포장·판매자 응대만 언급한 리뷰** — 제품 언급이 섞여 있으면 유지하되, 배송 관련 문장은 집계에서 무시.

---

## 분류 바구니 나누기

1. **일반 리뷰 vs 한달사용·재구매 리뷰** — 내구성 속성의 분모가 달라지므로 반드시 구분.
2. **옵션별 분리** — 물빠짐은 색상마다, 사이즈 정확도는 사이즈마다 다르므로 가능하면 옵션 단위로 집계.

---

## 분석 항목

### 1. 감성 집계
리뷰 단위가 아니라 **속성(aspect) 단위**로 긍정/부정을 센다.
- 예시: "핏은 예쁜데 보풀이 심해요" = 핏 긍정 1 + 보풀 부정 1

### 2. 키워드·문장 빈도
자주 언급되는 키워드와 문장을 횟수와 함께 정리.

### 3. 품질·내구성 1~5점
속성을 두 유형으로 나눠 계산한다.

| 유형 | 속성 | 계산법 | 분모 |
|---|---|---|---|
| 평가형 | 보온, 두께감, 신축성, 색감(화면 대비), 촉감 | 만족 비율 = 긍정 ÷ (긍정+부정) | 해당 속성 언급 리뷰 |
| 하자형(즉시 확인) | 실밥/봉제, 비침 | 문제 언급률 = 문제 언급 ÷ 분모 | 전체 유효 리뷰 |
| 하자형(사용 후 확인) | 수축, 물빠짐, 프린팅 크랙, 보풀, 넥 늘어남 | 문제 언급률 = 문제 언급 ÷ 분모 | 한달사용 + 재구매 리뷰만 |

### 4. 사이즈 분석
- **1차 소스**: 네이버 구조화 항목(정사이즈/커요/작아요) 집계.
- **2차 소스**: 텍스트 속 "키/몸무게 + 구매 사이즈 + 핏 평가" 조합.

### 5. 반품·교환
- 텍스트에서 "반품/교환했어요" 언급을 세고 사유를 분류.
- 플랫폼이 실제 반품률을 공개하지 않으므로 **"리뷰 기반 추정치"임을 결과물에 명시**.

### 6. 하자
- 하자형 속성 점수로 정량화.
- "한 번 세탁에 프린팅 전부 갈라짐" 같은 **심각 사례는 점수와 별개로 원문 인용 목록으로 남긴다**.

### 7. 실착용 경험
착용감, 활용처(데일리, 운동 등), 스타일링 언급을 정리.

---

## 점수 산정 방식

### 평가형 속성
만족 비율 = 긍정 언급 ÷ (긍정 언급 + 부정 언급)

| 점수 | 만족 비율 |
|---|---|
| 5점 | 90% 이상 |
| 4점 | 75~90% |
| 3점 | 60~75% |
| 2점 | 40~60% |
| 1점 | 40% 미만 |

### 하자형 속성
문제 언급률 = 문제 언급 리뷰 수 ÷ 유형별 분모 리뷰 수
- 즉시 확인형: 분모 = 전체 유효 리뷰
- 사용 후 확인형: 분모 = 한달사용 + 재구매 리뷰

| 점수 | 문제 언급률 |
|---|---|
| 5점 | 1% 이하 |
| 4점 | 1~3% |
| 3점 | 3~8% |
| 2점 | 8~15% |
| 1점 | 15% 초과 |

### 공통 규칙
- **점수 옆에 근거 숫자를 항상 표기.**
  - 예시: 보온성 4점 (언급 132건: 긍정 118 / 부정 14)
  - 예시: 보풀 3점 (한달사용리뷰 120건 중 언급 6건, 5.0%)
- **최소 표본 미만이면 점수 대신 "데이터 부족"으로 기재.**
  - 평가형: 해당 속성 언급 10건 미만
  - 하자형: 해당 분모 리뷰 수(전체 유효 리뷰 또는 한달사용+재구매 리뷰)가 30건 미만
- **구간 숫자는 제안 시작값** — 첫 분석 후 여러 상품의 점수 분포를 보고 보정한다. (전부 4~5점에 몰리면 구간이 후한 것)

---

## 최종 결과물

1. 긍정/부정 언급 수 (속성 단위)
2. 자주 언급되는 키워드·문장 + 횟수
3. 품질·내구성 속성별 1~5점 + 근거 숫자
4. 사이즈 평가 (정사이즈/작음/큼 비율 + 신체정보 조합)
5. 반품 언급률과 사유 (추정치 명시)
6. 심각 하자 사례 원문 목록
7. 실착용 경험 요약
8. 제외 리뷰 집계 (건수와 사유)
